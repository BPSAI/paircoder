# Objective

Show sandbox status in the main status command output when sandbox is configured or active.

# Files to Update

| File | Change |
|------|--------|
| `tools/cli/bpsai_pair/commands/core.py` | Add sandbox section to `status` command output |

# Implementation Plan

1. Detect sandbox state:
   - Check `PAIRCODER_SANDBOX` environment variable for active state
   - Check config for sandbox settings

2. Add sandbox section to status output:
   - Show active/inactive/not configured state
   - Show checkpoint ID if active
   - Show count of locked paths
   - Show network restriction status

3. Use color coding:
   - Green: Active and protecting
   - Yellow: Configured but not active
   - Gray: Not configured

4. Show protected paths summary (not full list)

# Acceptance Criteria

- [ ] Sandbox section appears in status output when sandbox enabled in config
- [ ] Shows: active/inactive, locked paths count, checkpoint ID
- [ ] Color coding: green (active), yellow (configured), gray (not configured)
- [ ] Network allowlist shown when active

# Implementation Notes

```python
# In core.py status command, add after current output:

def _show_sandbox_status(config):
    """Show sandbox status section."""
    sandbox = config.sandbox
    active = os.environ.get("PAIRCODER_SANDBOX") == "1"

    if not sandbox.enabled and not active:
        return  # Don't show if not configured

    console.print()
    console.print("[bold]Sandbox Status[/bold]")

    if active:
        checkpoint = os.environ.get("PAIRCODER_SANDBOX_CHECKPOINT", "none")
        console.print(f"   Mode: [green]ACTIVE[/green] (contained autonomy)")
        console.print(f"   Checkpoint: {checkpoint}")
    elif sandbox.enabled:
        console.print(f"   Mode: [yellow]CONFIGURED[/yellow] (not active)")
    else:
        console.print(f"   Mode: [dim]NOT CONFIGURED[/dim]")
        return

    dir_count = len(sandbox.locked_directories)
    file_count = len(sandbox.locked_files)
    console.print(f"   Locked Paths: {dir_count} directories, {file_count} files")

    net_count = len(sandbox.allow_network)
    console.print(f"   Network: Restricted ({net_count} domains allowed)")

    if sandbox.locked_directories or sandbox.locked_files:
        console.print()
        console.print("   [dim]Protected:[/dim]")
        for d in sandbox.locked_directories[:5]:
            console.print(f"   [dim]-[/dim] {d}")
        if len(sandbox.locked_directories) > 5:
            console.print(f"   [dim]... and {len(sandbox.locked_directories) - 5} more[/dim]")
```
